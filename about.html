"This is a story of an adventurer..."

The[dun]geon is a text-based rogue-like roleplaying game. Made interely in Python, game uses names, toponims, entities, weapons, objects and characters from the text files, so everything can be added/deleted/edited anytime anywhere. Try it out yourself! Folders are: "chargen", "storygen" and "eventgen". Rules of the game is in README file.

The game was made out of the boredom in a span of 1 month by a newbie developer, so feel free to judge my code in DMs. Other than that, I wanted to create original immersive universe for this game. Wiki for the game avaliable in the "World" button up there.

Anyone can contribute to it! Your questions, suggestions and comments are welcomed on the Discord server.
